# Windows Cheat Sheet

## Methods to shutdown machine

### Using Keyboard Shortcut

1. Press `Alt` + `F4`.

2. Select Shut down in the dialog box, then click "OK" to confirm.

### Via the Command Prompt

1. Press `Windows + S`, type "Command Prompt" in search and open it.

2. Type the following command:

   ```ps1
   shutdown /s /t 0
   ```
